In this application you can maintain your translation files.


All data is stored in the database.


On the download page it is possible to retrieve the translations you need in json-format. Just copy the code and paste it in your own .json file.
Another option is to load the required translations from the database into your application.


On the upload page you can copy the contents an existing .json translation file and load them into this application.
